Vodafone Chatbot Demo

This chatbot helps the customer find out whether he/she is entitled to a phone upgrade.
The chatbot has been implmented using the SAP Conversational AI platform and the Python Flask web framework.
The Flask application is hosted on github. The chatbot interface is hosted on Heroku.
