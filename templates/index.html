Telco Chatbot Demo

This chatbot can perform two tasks:

  • Look up whether the Customer is entitled to an iPhone upgrade. To this end, a simple business rule is used to inform the Chatbot's response: only one-, and two-year contracts can request an upgrade. Where an upgrade is possible, the Chatbot will share a link with the user for next steps on the Vodafone website. No link will be provided if the customer is not entitled to an upgrade.
  • Lookup the customer's monthly and total charges.

Some technical details about the Chatbot:

  • For the purposes of this Chatbot Demo, two datasets have been sourced from Kaggle. The first dataset, Telco Customer Churn, is used as the customer information data source. The second daaset, Customer Support on Twitter, is used as source for sentences used by SAP Recast AI to identify user intent.
  • uses Python Pandas for the backend data validation and lookup. Data is stored in memory. In a production environment this can be changed to interface with an actual database.
  • is deployed as a Single Page Application using Flask. Application files and code are stored on gitlab here.
