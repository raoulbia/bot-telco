Telco Chatbot Demo

This chatbot helps the customer find out whether he/she is entitled to a phone upgrade.
The chatbot has been implemented using the SAP Conversational AI platform and the Python Flask web framework.
The Flask application is hosted on github. The chatbot interface is hosted on Heroku.

Note: If you are re-visiting this page after having tested the demo chatbot for the first time, you may encounter
unresponsiveness. In this case please clear your browser cache and cookies or use your
browser in incognito mode.

The chatbot features include:

  • In-bot Customer ID confirmation
  • In-bot Customer ID format validation
  • Backend Customer ID Verification
  • Backend application of business rules to assess upgrade eligibility
  • Tailored product offer based on business rules
  • Fulfillment preference prompt
  • Leave feedback prompt
